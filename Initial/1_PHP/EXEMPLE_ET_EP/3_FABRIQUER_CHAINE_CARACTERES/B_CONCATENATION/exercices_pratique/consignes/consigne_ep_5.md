# Exercices pratiques : Concaténation, déclaration et assignation en PHP

## Objectif
Ces exercices vous permettront de pratiquer la **concaténation de chaînes de caractères**, ainsi que la **déclaration** et l'**assignation de variables** en PHP. Vous combinerez ces notions pour créer des messages dynamiques.

---

## Exercice 1 : Concaténation simple sans variables
1. Concaténez les chaînes `"Bonjour"` et `"le monde"` pour afficher le message : `"Bonjour le monde"`.

---

## Exercice 2 : Concaténation avec des variables
1. Créez une variable `$prenom` contenant votre prénom.
2. Créez une variable `$nom` contenant votre nom.
3. Concaténez-les pour afficher le nom complet.

---

## Exercice 3 : Concaténation avec des nombres
1. Créez une variable `$age` contenant un nombre.
2. Concaténez-la avec une chaîne de caractères pour afficher un message comme : `"J'ai X ans."`

---

## Exercice 4 : Assignation et concaténation
1. Créez une variable `$message` contenant la chaîne `"J'ai acheté "`.
2. Créez une variable `$quantite` contenant un nombre.
3. Créez une variable `$produit` contenant le nom d'un produit.
4. Concaténez ces variables pour afficher un message comme : `"J'ai acheté X Y."`

---

## Exercice 5 : Concaténation avec des opérations
1. Créez une variable `$nombre1` contenant la valeur `10`.
2. Créez une variable `$nombre2` contenant la valeur `20`.
3. Concaténez-les pour afficher un message comme : `"La somme de X et Y est Z."` (en calculant la somme).

---

## Exercice 6 (Bonus) : Concaténation complexe
1. Créez une variable `$ville` contenant le nom d'une ville.
2. Créez une variable `$pays` contenant le nom d'un pays.
3. Concaténez-les pour afficher un message comme : `"Je vis à X, en Y."`
4. Créer une variable qui contient la concaténation des élements suivants:
    - le message de l'exercice 1
    - le nom et prenom de l'exercice 2
    - l'age de l'exercice 3
5. Afficher la variable créée à la question précédente

---

## Instructions pour les exercices
- Ouvrez le fichier `/reponses/ep_5.php`.
- Écrivez votre code dans les espaces prévus (`^ ^ Écrivez au-dessus ^ ^`).
- Ajoutez vos propres instructions pour afficher les résultats (ex : `echo`, `var_dump`).
- Exécutez le fichier pour vérifier vos résultats.
